# 第3章：重复的牛奶

## 本章目标
苏念的重复引起林夜怀疑，他试图打破循环。老周的态度暗示他知道更多。林夜开始主动探寻便利店真相。

## POV
林夜（第一人称）

## 情绪基调
悬疑升级，从不安到主动探索

## 场景列表
1. [开场] 林夜开始计数——苏念已经连续第七天来买同一种牛奶，坐在同一个位子，看同一页书。他问老周："她是不是每天都来？"老周说："有些客人需要时间。" - POV:林夜 - 约600字
   - 核心动作：林夜记录苏念的来店规律
   - 情绪：好奇→较真

2. [发展] 林夜试着和苏念搭话，发现她的对话也是重复的——"我还有好多没背完""明天就考了""我妈说让我早点睡"。林夜试探："你记得昨天来过吗？"苏念茫然摇头 - POV:林夜 - 约1000字
   - 核心动作：林夜发现苏念被困在循环里，试图叫醒她
   - 情绪：同情→焦急

3. [转折] 林夜问老周"她到底怎么回事"，老周沉默很久，说了一句："你不能直接告诉她，她得自己想起来。"林夜追问"想起什么"，老周说："想起她已经不需要再考试了。" - POV:林夜 - 约800字
   - 核心动作：老周第一次透露便利店的"客人"有特殊之处
   - 情绪：震惊→决心（要弄清楚）

4. [结尾] 那晚打烊后，林夜没有回"家"（他也找不到家），而是在便利店里坐到天亮。他发现——天亮的时候，便利店的一切都变得模糊，像水中的倒影，然后消失了。他自己，也消失了 - POV:林夜 - 约600字
   - 悬念/衔接：林夜去哪了？为什么天亮后他也消失了？他到底是什么？

## 关键对话
- 「林夜」：「你记得昨天来过吗？」（试探）
- 「苏念」：「昨天？昨天我在图书馆啊……不对，我在哪？」（困惑/动摇）
- 「老周」：「她得自己想起来。你直接告诉她，她反而会害怕。」（经验/保护）
- 「林夜」：「老周，这家店到底是什么？」（追问）
- 「老周」：「……是一家便利店。卖东西的。」（回避）

## 伏笔记录
- 伏笔1：苏念说"昨天我在图书馆" → 她死在图书馆
- 伏笔2：天亮后林夜消失 → 他也不是活人
- 伏笔3：老周说"有些客人需要时间" → 他在说自己的经验
