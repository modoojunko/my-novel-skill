# 子夜便利店的规则

## 基本信息
- **类别**: 魔法/能力
- **来源**: 未知（存在时间远超百年）

## 核心规则

### 营业时间
- 每晚 00:00 准时出现，04:00 准时消失
- 出现地点：城市中某条老旧巷子的尽头
- 只有"需要被引导的亡魂"和"店员"能看到这家店

### 客人规则
1. 每个客人只能买一件东西——那件东西代表他们的执念
2. 买下东西不等于放下执念，只有真正放下才能离开
3. 如果天亮前没放下执念，客人会"重来"，第二天再来
4. 客人不会记得前一天的重复，但店员记得

### 店员规则
1. 店员由前任店员"选中"——只有死者才能当店员
2. 店员的职责：帮助客人认清执念、放下执念
3. 店员可以记住每一天，不会像客人一样重置
4. 店员不能主动告诉客人"你死了"——客人必须自己意识到

### 超度
- 当客人放下执念的那一刻，会看到一扇门
- 推开门就代表离开，不会再回来
- 超度一个人后，便利店会补充一件新货——代表新的客人即将到来

### 特殊现象
- 便利店里不会有日出，永远停留在凌晨
- 收银机的数字不是钱，是"已超度人数"
- 货架上的商品会自动补充，每件商品对应一个客人的执念
