

### 晚间更新（22:15）

新增 `update_specs` 模块，实现写作后自动检测并更新设定文档：

| 模块 | 文件 | 功能 |
|------|------|------|
| define | src/define.py | 人物卡和世界观设定管理 |
| update_specs | src/update_specs.py | 写作后自动检测新设定 |

#### 新增命令

```
story:define character --list       # 列出人物
story:define character 张三 --view   # 查看人物详情
story:define world --list           # 列出世界观
story:update-specs 5               # 分析第5章，检测新设定
story:update-specs 5 --auto         # 自动添加所有新设定
```
